### 使用方法：

直接下载已经打包好的发行版~

https://github.com/zdxiaoda/random-roll-call-system/releases

或者自己打包~

`git clone https://github.com/zdxiaoda/random-roll-call-system.git`

`cd random-roll-call-system `

`npm install`

`npm run make`
